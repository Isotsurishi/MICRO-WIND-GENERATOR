マイクロ風力（手回し）発電機の試作と実験記録
<br>
低風速でも発電可能な構造を目指す
<br>
その為にコアレスではなくコアのあるコイルでコギングを抑制し、磁気の漏れも抑制する構造とした
<br>

<br>

## 🔍 コイルコア材質による発電量の比較（コアなし = 1.0）

| コア材質               | 相対発電量（コアなし = 1.0） | 備考 |
|------------------------|------------------------------|------|
| コアなし（エアコア）     | 1.0                          | 渦電流なし。軽量だが磁束密度が低く、発電量は小さい。 |
| ソフトフェライト         | 2.5〜4.0                     | 高周波特性に優れ、渦電流損失が少ない。磁束密度は中程度。 |
| 圧粉磁性体（圧粉コア）   | 3.0〜5.0                     | 三次元磁束に強く、渦電流を抑えつつ高い磁束密度が得られる。 |
| 電磁鋼板（ケイ素鋼板）   | 4.0〜6.0                     | 高い磁束密度が得られるが、渦電流対策（積層など）が必要。低周波向き。 |
| SS400（軟鋼）           | 1.5〜2.5                     | 磁束密度は高いが、渦電流損失が大きく、発熱しやすい。加工性は良好。 |

Copilotでは上記のような結果になっていますが、コア無しではコイルが長いと磁束密度が届かないと思っています
<br>
現在、磁束密度を維持しつつコアとコギングを抑制する板の材料を猛者作中です
<br>
フェライトでは磁気飽和が速いためコギングが残ってると考えています
<br>
個人で入手不可能な材料を考えるより、他にできることは何か考えているところです
