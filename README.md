マイクロ風力（手回し）発電機の試作と実験記録

低風速でも発電可能な構造を目指す



Copilotで発電量をコア無しを「１」とした場合の比較

以下は、コイルにコアがない場合を「1」としたときの、各コア材質による相対的な発電量の目安（理論的な傾向）だよ

コア材質	相対発電量（コアなしを1とした場合）	

備考　コアなし（エアコア）	1.0	渦電流なし。軽量だが磁束密度が低く、発電量は小さい。

ソフトフェライト	2.5〜4.0	高周波特性に優れ、渦電流損失が少ない。磁束密度は中程度。

圧粉磁性体（圧粉コア）	3.0〜5.0	三次元磁束に強く、渦電流を抑えつつ高い磁束密度が得られる。

電磁鋼板（ケイ素鋼板）	4.0〜6.0	高い磁束密度が得られるが、渦電流対策（積層など）が必要。低周波向き。

SS400（軟鋼）	1.5〜2.5	磁束密度は高いが、渦電流損失が大きく、発熱しやすい。加工性は良好。

補足ポイント：

•	ソフトフェライトは高周波に強く、低損失だけど飽和磁束密度が低め（0.3〜0.5T程度）。

•	圧粉磁性体はフェライトと電磁鋼板の中間的な性質で、形状自由度が高く、渦電流も抑えられる。

•	電磁鋼板は高い磁束密度（1.5〜2.0T）を持つけど、積層構造にしないと渦電流が大きくなる。

•	SS400は安価で加工しやすいけど、磁気特性は電磁鋼板に劣るし、渦電流損失も大きい。
