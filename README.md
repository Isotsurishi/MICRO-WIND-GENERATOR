マイクロ風力（手回し）発電機の試作と実験記録
<br>
低風速でも発電可能な構造を目指す
<br><br>
発電機の構造はアキシャルギャップ型のコアコイルを使った単相交流発電機です
<br>
　１、コアにはアンテナ用フェライトロッドを使用
<br>
　２、アキシャル側の位置決めにリング状のネオジム磁石の反発を利用
<br>
　３、コギングを抑制するために磁性体の板を磁石とコイルの間に配置
<br>
　４、磁石のコイルと反対側の面から漏れる磁気を防ぐために隣接する磁石を磁性体で繋げた
<br>
　５、負荷に応じて発生する逆起電力を小さくするためにコイルは全て直列に結線した
<br>
今回の実験で使用したフェライトロッドの特製は不明ですが、透磁性は良いが磁気飽和しやすいためにコギングが残ったと考えています
<br>
実用化するには発電した電気を一度バッテリーに充電した後、出力するのが良いと考えています
